# AccioMates

AccioMates is a platform designed for **AccioJobs students** to connect, share opportunities, manage profiles, and chat with peers. This project allows students to **signup, login, post opportunities, and communicate** with other students seamlessly.

---

## Features

- **User Authentication**: Students can sign up and log in securely. Passwords are hashed and JWT is used for authentication.
- **Profile Management**: Users can view their profile information on the dashboard.
- **Opportunity Posts**: Students can post and explore job and project opportunities.
- **Chat Feature**: Real-time messaging between students for collaboration.
- **Responsive Design**: Fully responsive UI built with **React** and **Tailwind CSS**.
- **Protected Routes**: Users must be authenticated to access the dashboard.

---

## Tech Stack

- **Frontend**: React.js, React Router, Tailwind CSS, Axios  
- **Backend**: Node.js, Express.js, MongoDB, Mongoose  
- **Authentication**: JWT, bcrypt, cookies  
- **Deployment**: Backend on [Render](https://render.com), Frontend on Vite/React
